**FOSSBilling** is a free, open-source billing and client management system. It helps online service businesses automate invoicing, payments, and client communication. While it is popular with web hosting providers, it also works well for software licensing, digital downloads, and subscription services.

## What We Build

- **FOSSBilling**: the core self-hosted billing and client management application.
- **Documentation**: installation, upgrade, administration, and developer guides.
- **Extension directory**: a place to discover modules, payment gateways, server managers, and integrations.
- **Project infrastructure**: services that support release metadata, alerts, translations, and the public website.

## Getting Started

- Download the latest release from [fossbilling.org/downloads](https://fossbilling.org/downloads/).
- Read the [getting started guide](https://docs.fossbilling.org/getting-started/).
- Check the [system requirements](https://docs.fossbilling.org/getting-started/requirements/).
- Try the public [demo](https://fossbilling.org/demo).

## Community

FOSSBilling is developed in the open by contributors and users. You can help by reporting bugs, improving documentation, translating the interface, reviewing changes, building extensions, or supporting the project financially.

- Main repository: [FOSSBilling/FOSSBilling](https://github.com/FOSSBilling/FOSSBilling)
- Documentation: [docs.fossbilling.org](https://docs.fossbilling.org/)
- Forum: [forum.fossbilling.org](https://forum.fossbilling.org/)
- Discord: [fossbilling.org/discord](https://fossbilling.org/discord)
- Translations: [translate.fossbilling.org](https://translate.fossbilling.org/)
- Funding: [GitHub Sponsors](https://github.com/sponsors/FOSSBilling) or [Open Collective](https://opencollective.com/FOSSBilling)
